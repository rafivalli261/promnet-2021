Async & Await

Javascript is a single-threaded, non-blocking, asynchronus and concurrent language

Tread

Urutan eksekusi kode yang dapat dilakukan secara bebas / independen satu sama lain, artinya javascript hanya mampu melakukan satu hal pada satu waktu tertentu -> single thread, kalo multi nanti dibagi bagi

Blocking

Jika task 1 belum selesai maka task 2 tidak dieksekusi | non-blocking bisa tetap menjalankan task 2 walaupun task 1 belum selesai

Synchronus

biasa, normal flow | Asynchronus -> walaupun task 1 belum selesai bisa pindah ke task 2

asynchronus + single-threaded = Concurrency

asynchronus + multi-threaded = Parallelism
